# 花草百科全书 - UI Demo

**设计风格：** 深墨绿 AI Native

---

## 🎨 配色方案

### 主色系

| 名称 | 色值 | 用途 |
|------|------|------|
| **深墨绿** | `#0D1F12` | 背景、主色调 |
| **亮绿** | `#4CAF50` | 主按钮、强调色 |
| **白** | `#FFFFFF` | 主文字 |

### 辅助色

| 名称 | 色值 | 用途 |
|------|------|------|
| **淡绿灰** | `#A8C5B0` | 次要文字、描述 |
| **金黄** | `#FFD700` | 警告、提醒 |

---

## 📐 设计规范

### 尺寸

- 大植物视觉区：占屏幕 55%
- 悬浮 AI 按钮：直径 140rpx
- 主按钮高度：100rpx
- 卡片圆角：20-24rpx

### 字体

- 导航标题：34-36rpx
- 植物名称：44rpx
- 正文：30rpx
- 次要文字：28rpx
